SOBRE NOSOTROS

Okonomi Web Services es un sistema web diseñado para brindar un servicio de administracion de negocio totalmente completo, tomando en cuenta que una de sus principales ventajas es que no se necesita descargar abosulamente nada. Al usar Okonomi podras manejar tu negocio facil, seguro e inteligente.

MISION

  • Brindar el mejor servicio posible.
  • Hacer más fácil la labor de administrar un negocio.
  • Darle al usuario todo lo que necesita para manejar su negocio efectivamente e inteligentemente.

VISION

Poder llegar a ser la aplicación web número 1 en el mundo en el área de los negocios y para la administración del mismo, Brindando el mejor servicio y todas las herramientas necesaria para manejar tu negocio. A un precio asequible para todo los usuarios.

VALORES

  • Seguridad.
  • Trabajo en equipo.
